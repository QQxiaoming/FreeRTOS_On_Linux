# FreeRTOS_On_Linux

## 介绍

这是将FreeRTOS V10.2.1移植到linux平台的移植层代码，参考早期freertosV6版本的移植代码，并修复一些bug

## 编译

```shell
make
```

## 运行

```shell
./build/FreeRTOS_Posix.elf
```
